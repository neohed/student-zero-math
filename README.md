**Math for programmers. Built by a programmer. Learned in the open.**

This is a personal learning resource built on a simple premise: if you can write code, you are closer to understanding math than you think. Programmers use sophisticated math-adjacent abstractions every day — they just don't know it yet.

Every notebook in this repo is written to be understood by a working programmer with no formal math background. Concepts are built from intuition first, notation second. Theory is always followed by working code. This is an engineering resource, not an academic one.

→ For the full story, read [PHILOSOPHY.md](./PHILOSOPHY.md)

## What's Inside

### `fundamentals/`
Mathematical foundations built from the ground up, through the programmer's lens. Pre-algebra, algebra, calculus, linear algebra, geometry, complex numbers, and dynamical systems — each topic connected to concepts you already know from code.

### `books/`
Companion notebooks for key texts, including a series for *Sutskever's List* — a curated collection of landmark AI research papers. Each paper gets its own folder with a notebook and at least one working proof-of-concept project.

### `tools/`
Utilities for working with this repo. Planned: semantic search across notebooks.

---

## Proof-of-Concept Projects

Every research paper companion includes a small standalone PoC — real code, running in the real world, demonstrating what the math actually enables. Theory without implementation is incomplete.

---

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/neohed/student-zero-math.git
cd student-zero-math
pip install -r requirements.txt
```

Open any notebook in VS Code or JupyterLab and start reading. Each notebook is self-contained — no prior notebook in the series is required to follow along, though reading in order helps.
