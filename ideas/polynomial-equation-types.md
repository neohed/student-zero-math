# Polynomial Equation Types

Degree,Name,General form,Max real roots,Example
0,Constant,c = 0,all or none,5 = 5 (always true)
1,Linear,ax + b = 0,1,2x − 6 = 0
2,Quadratic,ax² + bx + c = 0,2,x² − 5x + 6 = 0
3,Cubic,ax³ + bx² + cx + d = 0,3,x³ − x = 0
4,Quartic,ax⁴ + … = 0,4,x⁴ − 1 = 0
5+,Quintic etc.,ax⁵ + … = 0,up to 5+,x⁵ + x + 1 = 0 (hard!)


**Visualizing**

Equation type,Highest power,Degree,Why the name?,Graph shape (in 2 variables)
Linear,x¹,1,Graphs as a straight line,Straight line
Quadratic,x²,2,Involves squaring the var,Parabola (U or upside-down U)
Cubic,x³,3,Involves cubing,S-curve or with a wiggle
Quartic,x⁴,4,Fourth power,W-shape or more turns
